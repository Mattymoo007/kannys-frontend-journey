## AI-Assisted Learning Tips

### Effective Prompts for Learning
- "Explain [concept] like I'm a beginner"
- "What's the difference between [A] and [B]?"
- "Why would I use [X] instead of [Y]?"
- "Show me a simple example of [concept]"
- "What are common mistakes when using [feature]?"

### Debugging with AI
- Share the error message and relevant code
- Describe what you expected vs what happened
- Ask "why" after getting a fix to understand it
